Hero Slider

A plug-in for jQuery with API for creating animated hero-style experiences.

  • In an area of fixed height, content rotates by slide-style transitions.
  • Navigation is provided to move "left" or "right" by mouse click or single-point touch swipe.
  • Activated state includes detailed content and additional navigation.
  • Autorotation is available by configuration
